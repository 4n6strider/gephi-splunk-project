Add a script.

Build a script by entering a splunk search and any arguments the search requires (the script will be automatically added to this script server)

    This script builder currently only supports up to 3 arguments. Arguments should be denoted in the splunk search by replacing the argument value by $$ARG1$$. The number following ARG should be a number 1-3 that corresponds to the ARGx values you enter in the form below.

    Example splunk search:

      search sourcetype=bank src_ip=$$ARG1$$ | dedup src_ip | table src_ip | iplocation src_ip | table Country, City, Region

    Example ARG1 value:

      123.123.123.123

    Example ARG2 and ARG3 values:

      Leave Blank
